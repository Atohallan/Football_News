Prophecy - An Online Betting Platform

Introduction

“Prophecy” is an Online Betting Platform. It can be used to guess the result of WorldCup / Tournament matches in a kind of prediction. The players compete in guessing the correct result of some kind of match. Soccer, Cricket, Baseball, Basketball, or even Yes/No questions like: will Brazil win the next WorldCup.
Admin can control match lock/unlock, ratio update, question , option add instantly where user don't need to website reload
It has strong SQL injection protection system which will keep away this system from hackers.
